## 使用方法

安装最新版本的NodeJS，克隆代码并且安装依赖

```shell script
+ git clone https://github.com/zhou-seven/SevenBlog.git
+ sudo npm -g yarn
+ yarn install
+ yarn serve
```

访问http://localhost:3000
